# FusioMR: Simulation and Data Analysis Code

This repository contains the full set of R scripts used for the simulation studies and real-data analyses presented in the FusioMR manuscript. 

* The codebase is organized to support transparency, reproducibility, and easy adaptation for related Mendelian Randomization (MR) workflows. These scripts serve as templates for applying FusioMR to new datasets or extending the model design.
